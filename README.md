## Requerimientos

- PHP >=5.6.4
- MySQL

## Instalación

**Git**
```shell
  git clone https://github.com/rafaelchirel/crud-simple-centro-medico.git
```

- La base de datos se encuentra en la sigueinte direccion `/crud-simple-centro-medico/db/crud.sql`
```php
	DB_HOST = 'localhost';
	DB_USER = "root";
	DB_PASS = "";
	DB_NAME = 'crud';
```
